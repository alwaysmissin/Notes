## 终端多路复用（tmux）
- 命令的基本使用：
	- `tmux a`：打开上一个session
	- `tmux new -s <session-name>`：新建一个名为`session-name`的`session`
- `ctrl-b`
	- `c`：新建一个窗口
	- `z`：全屏当前的pane
	- 方向键：切换pane
	- `,`：重命名当前窗口
> 配置教程：https://www.hamvocke.com/blog/a-guide-to-customizing-your-tmux-conf/
