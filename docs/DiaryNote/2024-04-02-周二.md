 # Zerotier 的基本使用
- 加入网络: `sudo zerotier-one join <network>`
- `zerotier-cli: missing port and zerotier-one. port not found in /var/lib/zerotier-one` 错误: 缺失端口或端口被占用, 在 `/var/lib/zerotier-one` 目录下创建一个文件, 配置端口信息: 
```
{
  "settings": {
    "primaryPort": 9994
  }
}
```
