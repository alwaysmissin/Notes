## 四、循环队列

1. 意义：防止了伪溢出现象导致一个队列变“短”
2. 操作
   1. 队头：`head = (head + 1) % MAX`
   2. 队尾：`tail = (tail + 1) % MAX`
   3. 队列长度：`(head + MAX - tail) % MAX`
3. 空\满
   1. 空：`head == tail`;
   2. 满：`(tail + 1) % MAX == head`
