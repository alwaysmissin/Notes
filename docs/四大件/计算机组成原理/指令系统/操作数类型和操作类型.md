# 操作数类型和操作类型
## 操作数类型
- 地址
	- 无符号整数
- 数字
	- 定点数、浮点数和十进制数
- 字符
	- 由于计算机在处理信息的过程中不能以简单的字符形式存储和传送，因此普遍采用ASCII码
- 逻辑数据
